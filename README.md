
# Employee Salary Management System (ESMS)

Employee Salary Management System (ESMS) is a Django-based web application designed to manage employees, attendance, leaves, salary structures, and payslips efficiently. It provides an organized way for HR/Admin to handle employee records and automate salary calculations.

## 🚀 Features

* Employee Management (Add, View)
* Attendance Tracking
* Leave Management & Approval
* Salary Structure Setup
* Automatic Salary Calculation
* Payslip Upload & View

## 🛠️ Tech Stack

* Django
* SQLite
* HTML/CSS (Templates)

## ⚙️ Installation

```bash
git clone https://github.com/your-username/Employee_Salary.git
cd Employee_Salary
python -m venv env
env\Scripts\activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

## 📌 Usage

* Admin/HR can manage employees and salary
* Track attendance and leaves
* Generate salaries and upload payslips
